## Что можно сохранить

- **Классификатор 01–99** (Приём, Перевод, …) — align с PO taxonomy / Corpsite.
- **Принцип «классификация по событию, не по году»** — для hr-head и архивариуса.
- **Состав реквизитов электронной карточки** — основа для glossary в UG.
- **Модель двуязычия** (одна карточка, два текста) — соответствует UDE/PO direction.
- **План работ 1–5** — roadmap note для hr-head.

## Что устарело / осторожно

- Статус **Черновик 0.1** — не утверждённая политика.
- **REGISTERED** в реквизитах как обязательный статус — сверить с упрощённым lifecycle (SIGNED = завершение).
- Нет привязки к **текущему UI** Personnel Orders.
