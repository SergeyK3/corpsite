# PO-002 Классификация приказов по личному составу

**Статус:** Черновик  
**Версия:** 0.1

## Назначение

Документ определяет классификацию кадровых событий и соответствующих им приказов по личному составу для системы Corpsite.

Основной принцип: **классифицируются кадровые события, а не документы**. Приказ является юридическим оформлением одного или нескольких кадровых событий.

---

# 1. Классификация кадровых событий

## Трудовые отношения
- Прием
- Увольнение

## Должностные изменения
- Назначение
- Перевод

## Изменение условий труда
- Изменение ставки
- Совмещение
- Совместительство
- Доплата

## Отпуска и отсутствия
- Трудовой отпуск
- Часть трудового отпуска
- Отпуск без содержания
- Декретный отпуск
- Отпуск по уходу за ребенком
- Возврат из отпуска
- Командировка

- Присвоение ученой степени
- Изменение сведений об образовании
- Изменение стажевой группы

## Персональные данные
- Изменение персональных данных

## Временное исполнение обязанностей

## Управление приказами
- Изменение приказа
- Отмена приказа

---

# 2. Связь с шаблонами приказов

Одному кадровому событию соответствует один или несколько шаблонов приказов.

Система должна поддерживать:

- один приказ → одно событие;
- один приказ → несколько событий;
- один приказ → несколько сотрудников.

---

# 3. Результаты анализа пилотного архива

Выделены основные группы:

- прием;
- увольнение;
- перевод;
- изменение условий труда;
- отпуска;
- квалификация;
- изменение персональных данных;
- временное исполнение обязанностей;
- управление приказами.

Кадровые отчеты должны храниться отдельно от архива приказов.
