# 1. Назначение

Документ фиксирует базовую терминологию кадрового контура Corpsite.

Цель — исключить смешение понятий «кадровое событие», «приказ», «документ», «согласование», «состояние сотрудника» и «изменение».

---

# 2. Термины

## Employee

Сотрудник, представленный в кадровом контуре Corpsite.

## Personnel State

Текущее расчетное кадровое состояние сотрудника.

Формируется на основании действующих кадровых событий.

## Personnel Event

Кадровое событие.

Бизнес-сущность, описывающая изменение кадрового состояния сотрудника.

Примеры:

- прием;
- перевод;
- увольнение;
- отпуск;
- изменение ставки;
- доплата;
- присвоение категории;
- изменение персональных данных.

## Change Set

Набор изменений, принадлежащий одному кадровому событию.

## Personnel Change

Атомарное изменение одного атрибута.

Пример:

```text
field: position
old: врач
new: заведующий отделением
```

## Personnel Order

Приказ по личному составу.

Юридический документ, оформляющий одно или несколько кадровых событий.

## Order Item

Пункт приказа.

Связывает один пункт приказа с конкретным кадровым событием и сотрудником.

## Order Document

Файловое представление приказа.

Примеры:

- DOCX KZ;
- PDF KZ;
- DOCX RU;
- PDF RU;
- скан.

## Approval Route

Маршрут согласования проекта приказа или кадрового события.

## Visa / Approval

Виза или согласование.

Действие согласующего лица до подписания приказа.

## Signatory

Подписант приказа.

Лицо, уполномоченное подписать приказ.

## Preparer

Исполнитель приказа.

Сотрудник, подготовивший проект приказа.

## Temporary Assignment

Временное исполнение обязанностей или замещение.

Самостоятельное кадровое событие, которое может быть связано с отпуском, командировкой, болезнью или вакансией.

## Allowance

Доплата.

Финансовое условие, которое может быть самостоятельным событием или связанным событием.

## Personnel Report

Кадровый отчет.

Не является приказом и должен храниться отдельно от архива приказов.

# 3. Ключевые разграничения

## Personnel Event vs Personnel Order

Personnel Event — что произошло с сотрудником.  
Personnel Order — каким документом это оформлено.

## Personnel State vs Personnel Event

Personnel State — текущее состояние.  
Personnel Event — изменение, которое привело к состоянию.

## Order Document vs Personnel Order

Personnel Order — бизнес-документ.  
Order Document — файл: DOCX, PDF, скан.

## Approval vs Personnel Event

Approval — действие согласования.  
Personnel Event — кадровое изменение.

---

# 4. Базовая формула

```text
Personnel Event changes Personnel State.
Personnel Order legally registers Personnel Event.
Order Document represents Personnel Order.
Approval validates Personnel Order before signing.
```
