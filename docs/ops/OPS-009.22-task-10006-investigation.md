### B.2 Telegram delivery evidence

VPS query on delivery trail (read-only):

```sql
SELECT
    e.audit_id,
    e.event_type,
    e.created_at AS event_at,
    d.channel,
    d.status AS delivery_status,
    d.sent_at,
    rec.user_id,
    u.login AS recipient_login
FROM public.task_events e
LEFT JOIN public.task_event_deliveries d ON d.audit_id = e.audit_id
LEFT JOIN public.users u ON u.user_id = (
    SELECT r.user_id FROM public.task_event_recipients r
    WHERE r.audit_id = e.audit_id LIMIT 1
)
WHERE e.task_id = 10006
ORDER BY e.created_at, d.channel;
```

```sql
SELECT
    t.task_id, t.title, ts.code AS status_code,
    t.task_kind, t.source_kind, t.regular_task_id,

    ui.login AS initiator_login, er.code AS executor_role_code
```
